# AI Відео з Фотографій - Вебсайт

Професійний вебсайт для послуги створення AI-відео з фотографій на замовлення.

## Технології

- Next.js 14
- React 18
- TypeScript
- CSS Modules
- Telegram Bot API (для сповіщень про замовлення)

## Локальний запуск

```bash
npm install
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000) у браузері.

## Налаштування Telegram бота

Для отримання сповіщень про нові замовлення через Telegram:

### Крок 1: Створіть бота в Telegram

1. Відкрийте Telegram і знайдіть `@BotFather`
2. Надішліть команду `/newbot`
3. Дотримуйтесь інструкцій для створення бота
4. Скопіюйте токен бота (він виглядає як: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Крок 2: Отримайте Chat ID

1. Надішліть будь-яке повідомлення вашому новому боту
2. Відкрийте у браузері: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
3. Знайдіть у відповіді `"chat":{"id": ЧИСЛО}`
4. Скопіюйте це число - це ваш Chat ID

### Крок 3: Налаштуйте змінні середовища

Створіть файл `.env.local` в кореневій папці проекту:

```bash
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
```

### Для Vercel

У налаштуваннях проекту на Vercel (Settings > Environment Variables) додайте:
- `TELEGRAM_BOT_TOKEN` - токен вашого бота
- `TELEGRAM_CHAT_ID` - ваш Chat ID

## Як працює інтеграція

Коли відвідувач заповнює та надсилає форму замовлення:

1. Дані форми відправляються на API `/api/submit-form`
2. API форматує повідомлення з усіма деталями замовлення
3. Повідомлення надсилається у Telegram через Bot API
4. Ви отримуєте миттєве сповіщення з інформацією про клієнта

Формат повідомлення включає:
- Ім'я клієнта
- Телефон та email
- Тип події (весілля, день народження, ювілей, тощо)
- Кількість відео
- Додаткова інформація
- Дата та час замовлення

## Деплой на Vercel

1. Підключіть репозиторій до Vercel
2. Vercel автоматично визначить Next.js проект
3. Додайте змінні середовища для Telegram
4. Деплой відбудеться автоматично

Або використайте Vercel CLI:

```bash
npm i -g vercel
vercel
```
