# AI Відео з Фотографій - Вебсайт

Професійний вебсайт для послуги створення AI-відео з фотографій на замовлення.

## Технології

- Next.js 14
- React 18
- TypeScript
- CSS Modules

## Локальний запуск

```bash
npm install
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000) у браузері.

## Деплой на Vercel

1. Підключіть репозиторій до Vercel
2. Vercel автоматично визначить Next.js проект
3. Деплой відбудеться автоматично

Або використайте Vercel CLI:

```bash
npm i -g vercel
vercel
```
