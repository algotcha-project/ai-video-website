# AI Відео з Фотографій - Вебсайт

Професійний вебсайт для послуги створення AI-відео з фотографій на замовлення.

## Технології

- Next.js 14
- React 18
- TypeScript
- CSS Modules

## Локальний запуск

```bash
npm install
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000) у браузері.

## Telegram інтеграція

Форма замовлення автоматично відкриває Telegram з готовим повідомленням для @oleg030696.

Коли відвідувач заповнює форму і натискає "Відправити в Telegram":
1. Відкривається Telegram з попередньо заповненим повідомленням
2. Повідомлення містить всю інформацію про замовлення
3. Клієнту залишається лише надіслати повідомлення

## Деплой на Vercel

1. Підключіть репозиторій до Vercel
2. Vercel автоматично визначить Next.js проект
3. Деплой відбудеться автоматично

Або використайте Vercel CLI:

```bash
npm i -g vercel
vercel
```
