# Інструкції для деплою на Vercel

## Швидкий деплой (5 хвилин)

### Варіант 1: Через веб-інтерфейс Vercel (Найпростіший)

1. Перейдіть на [vercel.com](https://vercel.com)
2. Зареєструйтеся або увійдіть (можна через GitHub)
3. Натисніть "Add New Project"
4. Підключіть GitHub репозиторій (або завантажте папку проєкту)
5. Vercel автоматично визначить Next.js
6. Натисніть "Deploy"
7. Готово! Отримаєте URL типу: `your-project.vercel.app`

### Варіант 2: Через Vercel CLI

Якщо у вас встановлений Node.js:

```bash
npm i -g vercel
cd C:\Users\HDUser\ai-video-website
vercel
```

Слідуйте інструкціям в терміналі.

### Варіант 3: Через GitHub

1. Створіть репозиторій на GitHub
2. Завантажте файли проєкту:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin YOUR_GITHUB_REPO_URL
   git push -u origin main
   ```
3. Підключіть репозиторій до Vercel
4. Автоматичний деплой!
